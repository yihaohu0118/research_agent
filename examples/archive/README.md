# Archived Examples

These configs are kept for reference, ablations, and older demos. They are not
part of the primary BFCL co-evolution experiment matrix.

- `bfcl_patch_ablations/`: isolated T-Patch, A-Patch, E-Patch, and S-Patch rows.
- `no_coevo_diagnostics/`: older full-TAES, warmup, and early-stop BFCL diagnostics.
- `coevo_paused/`: coevolution configs temporarily paused while no-coevo
  miss-function experiments are run.
- `legacy/`: generic `basic`/`overall` examples and runner scripts.
- `game/`: Avalon and Diplomacy examples.

For the current environment/model co-evolution story, use the top-level BFCL
configs in `examples/`.
